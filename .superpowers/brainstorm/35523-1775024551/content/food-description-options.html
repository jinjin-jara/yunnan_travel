음식 설명 표시 방식 — 어떤 게 좋으세요?

현재 일정표에는 기화계(汽锅鸡), 미선(米线) 같은 낯선 음식 이름만 있습니다. 설명을 어떻게 추가할지 골라주세요.

A

항상 보이는 설명 태그

음식 이름 바로 아래 작은 한 줄 설명이 항상 표시됩니다. 호버 없이 바로 읽을 수 있어 모바일에서도 편리합니다.

점심 — 기화계(汽锅鸡) 🍲 증기로 천천히 쪄낸 통닭 요리 · 윈난 대표 보양식
B

기존 팝오버에 설명 추가

현재 지도·검색 링크가 뜨는 말풍선 안에 음식 설명을 함께 넣습니다. 클릭 한 번으로 설명과 링크를 한꺼번에 볼 수 있습니다.

기화계(汽锅鸡) ▾
🍲 증기로 천천히 쪄낸 통닭 요리. 맑은 국물이 특징인 윈난 대표 보양식.
🗺 구글지도 · 바이두지도 · 검색
C

클릭 시 인라인 펼침

음식 이름을 클릭하면 바로 아래 설명이 부드럽게 펼쳐집니다. 보고 싶을 때만 열어볼 수 있어 글이 복잡해 보이지 않습니다.

기화계(汽锅鸡) ℹ️
🍲 증기압으로 천천히 쪄낸 통닭 요리. 맑고 진한 국물이 특징이며, 윈난성을 대표하는 보양 음식입니다. 주로 한약재를 함께 넣어 끓입니다.
D

음식 전용 카드 (식사 항목만)

식사 항목에만 별도 음식 카드를 사용합니다. 사진·설명·링크를 한 카드에 담아 여행 가이드북처럼 보입니다. 가장 시각적이지만 구현이 복잡합니다.
